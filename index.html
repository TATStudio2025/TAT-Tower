เกี่ยวกับเกม TAT Tower

TAT Tower เป็นเกมแนว Strategy Defense ที่ท้าทายให้คุณวางแผนและป้องกันฐานทัพของคุณจากศัตรูที่มาโจมตี จุดเด่นของเกม: • ใช้กลยุทธ์ในการป้องกันฐาน • เล่นฟรี 100% ไม่มีโฆษณา • พัฒนาโดยทีมงานไทย • รองรับหลายแพลตฟอร์ม (Android, Linux) อยู่ในช่วง Early Access - เรายินดีรับฟังความคิดเห็นและข้อเสนอแนะจากผู้เล่น

คำถามที่พบบ่อย

TAT Tower เล่นฟรีจริงหรือไม่?

ใช่ เกมเล่นฟรี 100% ไม่มีการซื้อในเกม ไม่มีโฆษณา สามารถดาวน์โหลดและเล่นได้เลยทันที

ต้องใช้อินเทอร์เน็ตในการเล่นหรือไม่?

ต้องใช้ TAT Tower เล่นได้เฉพาะออนไลน์เท่านั้น

วิธีติดตั้งบน Linux?

ดาวน์โหลดไฟล์ .zip แล้วแตกไฟล์ แล้วรันไฟล์ executable (Linux)

เกมมีภาษาไทยหรือไม่?

เกมพัฒนาโดยทีมไทยและรองรับภาษาไทย รวมถึงภาษาอังกฤษด้วย

จะรายงานบั๊กหรือแนะนำฟีเจอร์ได้อย่างไร?

สามารถรายงานได้ทาง GitHub Issues หรือส่งอีเมลมาที่ tattower@timeandtime.online
