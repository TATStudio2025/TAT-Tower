ทีม TAT Studio กลับหน้าหลัก

นโยบายความเป็นส่วนตัว

อัพเดทล่าสุด: 7 กุมภาพันธ์ 2026

1. ข้อมูลทั่วไป

TAT Studio ("เรา", "ของเรา") ให้ความสำคัญกับความเป็นส่วนตัวของผู้ใช้งาน ("ท่าน", "ผู้เล่น") เกม TAT Tower นโยบายความเป็นส่วนตัวนี้อธิบายวิธีที่เราจัดการกับข้อมูลของท่านเมื่อใช้งานเกมของเรา

สรุปสำคัญ: เกม TAT Tower ไม่มีการเก็บรวบรวม จัดเก็บ หรือส่งข้อมูลส่วนบุคคลใดๆ ของผู้เล่นไปยังเซิร์ฟเวอร์ภายนอก เกมทำงานแบบออฟไลน์ทั้งหมด

2. ข้อมูลที่เราไม่เก็บรวบรวม

เกม TAT Tower ได้รับการออกแบบให้เคารพความเป็นส่วนตัวของท่าน เราไม่มีการเก็บรวบรวมข้อมูลดังต่อไปนี้:

  • ข้อมูลส่วนบุคคล (ชื่อ, อีเมล, เบอร์โทรศัพท์)
  • ข้อมูลตำแหน่งที่อยู่ (GPS, Location data)
  • ข้อมูลอุปกรณ์ (Device ID, IMEI, MAC Address)
  • ข้อมูลการใช้งาน (Analytics, Usage data)
  • ข้อมูลการเชื่อมต่ออินเทอร์เน็ต (IP Address)
  • ข้อมูลจากบัญชีโซเชียลมีเดีย
  • ข้อมูลการชำระเงิน (ไม่มีระบบซื้อในเกม)

3. ข้อมูลที่จัดเก็บในเครื่องของท่าน

เกม TAT Tower จะจัดเก็บข้อมูลบางอย่างในเครื่องของท่าน (Local Storage) เพื่อการทำงานของเกมเท่านั้น:

  • ความคืบหน้าในเกม (Level, Score)
  • การตั้งค่าเกม (ระดับเสียง, ภาษา)
  • สถิติการเล่น

ข้อมูลเหล่านี้:

  • จัดเก็บในอุปกรณ์ของท่านเท่านั้น
  • ไม่ถูกส่งไปยังเซิร์ฟเวอร์ใดๆ
  • สามารถลบได้โดยการถอนการติดตั้งเกม

4. การเชื่อมต่ออินเทอร์เน็ต

เกม TAT Tower สามารถเล่นได้แบบออฟไลน์ทั้งหมด ไม่จำเป็นต้องเชื่อมต่ออินเทอร์เน็ต เกมไม่มีการ:

  • ส่งหรือรับข้อมูลผ่านอินเทอร์เน็ต
  • เชื่อมต่อกับเซิร์ฟเวอร์ภายนอก
  • แสดงโฆษณาออนไลน์
  • ติดตามพฤติกรรมการใช้งาน

5. การใช้บริการของบุคคลที่สาม

เกม TAT Tower ไม่มีการใช้บริการวิเคราะห์ข้อมูล (Analytics) หรือบริการโฆษณาจากบุคคลที่สาม ไม่มี SDK หรือ Library ใดๆ ที่เก็บรวบรวมข้อมูลของท่าน

6. ความปลอดภัยของข้อมูล

แม้ว่าเกมของเราจะไม่มีการเก็บรวบรวมข้อมูลส่วนบุคคล เราก็ยังคงให้ความสำคัญกับความปลอดภัย:

  • ไม่มีการส่งข้อมูลผ่านเครือข่าย
  • ข้อมูลเกมถูกจัดเก็บในเครื่องอย่างปลอดภัย
  • ไม่มีช่องทางสำหรับการเข้าถึงข้อมูลจากภายนอก

7. สิทธิ์ของผู้ใช้งาน

ท่านมีสิทธิ์ในการ:

  • เล่นเกมโดยไม่ต้องกังวลเรื่องข้อมูลส่วนบุคคล
  • ลบข้อมูลเกมในเครื่องได้ทุกเมื่อผ่านการถอนการติดตั้ง
  • ใช้งานเกมแบบออฟไลน์ทั้งหมด

8. เด็กและเยาวชน

เกม TAT Tower เหมาะสำหรับผู้เล่นทุกวัย เนื่องจากเกมไม่มีการเก็บรวบรวมข้อมูลใดๆ จึงปลอดภัยสำหรับเด็กและเยาวชน ไม่จำเป็นต้องได้รับความยินยอมจากผู้ปกครอง

9. การเปลี่ยนแปลงนโยบาย

เราอาจปรับปรุงนโยบายความเป็นส่วนตัวนี้เป็นครั้งคราว หากมีการเปลี่ยนแปลงที่สำคัญ เราจะแจ้งให้ทราบผ่านทาง:

  • การอัพเดทหน้าเว็บไซต์นี้
  • ประกาศใน Release Notes ของเกม
  • GitHub Repository
